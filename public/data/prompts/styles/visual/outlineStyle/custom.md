# Custom Prompt

## 中文版本 (zh-CN)

请使用 TailwindCSS 创建一个 Outline Style 页面：主体由线框/描边构成，填充极少。包含 Hero、分栏信息卡、流程/列表区、CTA。使用 1-2px 描边、线性图标、几何无衬线字体，背景可为浅灰或深色，保证高对比与清晰层级。


---

## English Version (en-US)

Use TailwindCSS to build an Outline Style page with stroke-first elements and minimal fills. Include a hero, info cards, a flow/list section, and CTAs. Employ 1–2px strokes, linear icons, geometric sans type on light-gray or dark backgrounds with clear hierarchy and contrast.
