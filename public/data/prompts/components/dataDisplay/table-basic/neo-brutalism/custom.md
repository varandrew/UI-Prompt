# Neo Brutalism Basic Table

## 中文版本 (zh-CN)

請創建一個新粗野主義風格的基礎表格組件。

### 設計理念
新粗野主義表格以大膽、張揚的視覺語言打破傳統 UI 的溫和審美。通過粗重的邊框、鮮明的撞色和刻意的不完美感，傳達出反叛精神和獨特個性。這種設計拒絕過度修飾，擁抱原始的視覺衝擊力，為年輕化和創意化的產品注入強烈的品牌個性。

### 視覺特徵
- **配色方案**：使用高飽和度的撞色組合，如鮮黃、亮粉、天藍等，配合純黑和純白形成強烈對比
- **邊框設計**：粗重的純黑邊框（2-4px），嚴格的方角設計，拒絕任何圓角修飾
- **陰影效果**：硬邊偏移陰影，無模糊效果，營造強烈的立體感和層次感
- **排版風格**：粗體字重，較大的字號，強調視覺衝擊力
- **區塊分割**：大膽的色塊分割，表頭和數據行使用對比強烈的背景色

### 交互體驗
- **懸停效果**：行懸停時背景色快速切換或陰影加深，提供強烈的視覺反饋
- **點擊反饋**：明確的視覺狀態變化，可能包含輕微的位移效果
- **選中狀態**：使用高對比度的顏色標記選中行，確保一目了然
- **動畫設計**：快速直接的過渡動畫，避免過度平滑的緩動效果

### 適用場景
- 創意設計類產品和藝術平台
- 年輕化、潮流化的品牌應用
- 音樂、遊戲等娛樂行業的數據展示
- 需要強烈視覺衝擊和品牌個性的產品
- 反傳統、追求獨特性的創新項目

## English Version (en-US)

Please create a neo brutalism style basic table component.

### Design Philosophy
Neo brutalism table uses bold and expressive visual language to break traditional UI's moderate aesthetics. Through heavy borders, striking color clashes, and intentional imperfections, it conveys rebellious spirit and unique personality. This design rejects excessive embellishment, embracing raw visual impact to inject strong brand personality into youthful and creative products.

### Visual Characteristics
- **Color Scheme**: Uses highly saturated contrasting colors like bright yellow, hot pink, cyan, combined with pure black and white for strong contrast
- **Border Design**: Heavy pure black borders (2-4px) with strict square corners, rejecting any rounded corner decoration
- **Shadow Effects**: Hard-edge offset shadows without blur, creating strong three-dimensional feel and layering
- **Typography Style**: Bold font weight with larger font sizes emphasizing visual impact
- **Block Division**: Bold color block divisions with strongly contrasting background colors between headers and data rows

### Interaction Experience
- **Hover Effects**: Quick background color switching or shadow deepening on row hover, providing strong visual feedback
- **Click Feedback**: Clear visual state changes, possibly including slight displacement effects
- **Selected State**: Uses high-contrast colors to mark selected rows, ensuring at-a-glance recognition
- **Animation Design**: Fast and direct transition animations, avoiding overly smooth easing effects

### Use Cases
- Creative design products and art platforms
- Youthful and trendy brand applications
- Data display in entertainment industries like music and gaming
- Products requiring strong visual impact and brand personality
- Anti-traditional innovative projects pursuing uniqueness
