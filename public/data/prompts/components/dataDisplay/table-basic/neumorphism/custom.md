# Neumorphism Basic Table

## 中文版本 (zh-CN)

請創建一個新擬態風格的基礎表格組件。

### 設計理念
新擬態表格通過巧妙的光影設計，創造出元素從背景中浮起或嵌入的視覺錯覺。使用同色系的明暗雙重陰影系統，營造柔和的立體效果和觸覺質感。這種設計既保留了扁平化的簡潔，又增添了擬物化的溫暖觸感，為數據展示帶來獨特的視覺體驗和舒適感。

### 視覺特徵
- **背景基調**：淺灰色背景系統，與表格元素顏色接近但略有差異
- **浮雕陰影**：雙重陰影系統（明暗配合），營造柔和的凸起或凹陷效果
- **配色方案**：單色調系統，使用灰階色系，通過陰影和高光創造層次
- **圓角設計**：中等圓角，柔化邊緣，增強親和力和柔軟觸感
- **表頭設計**：輕微凸起效果的表頭，與凹陷的數據行形成對比

### 交互體驗
- **懸停狀態**：行懸停時陰影深度變化，營造"按下"的觸感反饋
- **選中狀態**：使用凹陷陰影效果標記選中行，模擬真實按下效果
- **排序指示**：凸起的排序按鈕，配合微妙的陰影變化
- **響應式設計**：保持一致的陰影系統和觸覺反饋，確保各設備體驗統一

### 適用場景
- 追求獨特視覺風格的現代應用界面
- 智能家居和物聯網設備控制面板
- 需要柔和、舒適視覺體驗的健康醫療應用
- 數據管理工具和個人效率應用
- 淺色主題下的數據可視化系統

## English Version (en-US)

Please create a neumorphism style basic table component.

### Design Philosophy
Neumorphism table creates visual illusions of elements floating above or embedded into the background through clever light and shadow design. Using a dual shadow system of light and dark in the same hue, it creates soft three-dimensional effects and tactile texture. This design retains the simplicity of flat design while adding the warm tactile quality of skeuomorphism, bringing unique visual experience and comfort to data display.

### Visual Characteristics
- **Background Base**: Light gray background system with colors similar to but slightly different from table elements
- **Embossed Shadows**: Dual shadow system (light and dark combined) creating soft raised or recessed effects
- **Color Scheme**: Monochromatic system using grayscale colors, creating layers through shadows and highlights
- **Rounded Design**: Medium border radius softening edges, enhancing approachability and soft touch
- **Header Design**: Slightly raised header contrasting with recessed data rows

### Interaction Experience
- **Hover State**: Shadow depth changes on row hover, creating "press down" tactile feedback
- **Selected State**: Uses recessed shadow effect to mark selected rows, simulating real press-down effect
- **Sort Indicators**: Raised sort buttons with subtle shadow changes
- **Responsive Design**: Maintains consistent shadow system and tactile feedback ensuring unified experience across devices

### Use Cases
- Modern application interfaces pursuing unique visual style
- Smart home and IoT device control panels
- Health and medical applications requiring soft, comfortable visual experience
- Data management tools and personal productivity applications
- Data visualization systems under light themes
