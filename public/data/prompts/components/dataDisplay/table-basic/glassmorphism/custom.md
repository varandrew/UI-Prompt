# Glassmorphism Basic Table

## 中文版本 (zh-CN)

請創建一個玻璃擬態風格的基礎表格組件。

### 設計理念
玻璃擬態表格通過半透明毛玻璃效果，將數據展示提升為一種視覺藝術體驗。層疊的半透明元素在漸變背景上營造出夢幻般的深度感和空間層次，讓表格不再是冰冷的數據容器，而是充滿現代感和科技感的視覺界面，為用戶帶來輕盈、優雅的數據瀏覽體驗。

### 視覺特徵
- **玻璃效果**：半透明背景配合背景模糊濾鏡，實現標誌性的毛玻璃質感
- **邊框光澤**：細膩的半透明白色邊框，模擬玻璃邊緣的反光效果
- **配色方案**：漸變色背景（如紫藍、藍綠漸變），表格使用不同透明度的白色層疊
- **陰影層次**：柔和的多層陰影，增強浮空感和立體深度
- **表頭設計**：較高透明度的表頭，與數據行形成層次對比

### 交互體驗
- **懸停效果**：行懸停時增強背景透明度和模糊效果，配合微光暈染
- **選中狀態**：使用高亮的玻璃效果標記選中行，保持整體風格統一
- **排序指示**：半透明的排序箭頭，與玻璃質感自然融合
- **響應式適配**：移動端優化模糊效果以提升性能，保持視覺風格一致性

### 適用場景
- 現代科技產品和 SaaS 平台的數據面板
- 加密貨幣和區塊鏈應用的數據展示
- 創意設計和數字藝術平台
- 需要營造未來感和科技氛圍的應用界面
- 暗色主題下的數據可視化系統

## English Version (en-US)

Please create a glassmorphism style basic table component.

### Design Philosophy
Glassmorphism table elevates data display into a visual art experience through semi-transparent frosted glass effects. Layered semi-transparent elements create dreamlike depth and spatial hierarchy on gradient backgrounds, transforming tables from cold data containers into modern and tech-savvy visual interfaces, bringing users a light and elegant data browsing experience.

### Visual Characteristics
- **Glass Effect**: Semi-transparent background with backdrop blur filters achieving signature frosted glass texture
- **Border Shine**: Delicate semi-transparent white borders simulating glass edge reflection effects
- **Color Scheme**: Gradient backgrounds (purple-blue, blue-green gradients) with tables using different transparency levels of white layering
- **Shadow Layers**: Soft multi-layered shadows enhancing floating sensation and three-dimensional depth
- **Header Design**: Higher transparency headers creating layered contrast with data rows

### Interaction Experience
- **Hover Effects**: Enhanced background transparency and blur effects on row hover with subtle glow diffusion
- **Selected State**: Uses highlighted glass effect to mark selected rows, maintaining overall style unity
- **Sort Indicators**: Semi-transparent sort arrows naturally blending with glass texture
- **Responsive Adaptation**: Optimized blur effects on mobile for performance while maintaining visual style consistency

### Use Cases
- Modern tech products and SaaS platform data panels
- Cryptocurrency and blockchain application data displays
- Creative design and digital art platforms
- Application interfaces requiring futuristic and tech atmosphere
- Data visualization systems under dark themes
