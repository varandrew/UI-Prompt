# Bootstrap 5 Basic Table

## 中文版本 (zh-CN)

請創建一個 Bootstrap 5 風格的基礎表格組件。

### 設計理念
Bootstrap 5 表格延續了 Bootstrap 框架實用主義的設計傳統，提供開箱即用的專業外觀和標準化的組件結構。通過成熟的設計規範和豐富的樣式變體，為開發者提供快速構建數據表格的解決方案，確保跨項目的一致性和團隊協作的效率。

### 視覺特徵
- **配色方案**：使用 Bootstrap 語義化配色系統，Bootstrap 藍為主色，配合灰階系統
- **條紋設計**：可選的斑馬條紋行，交替使用白色和淺灰背景，提升數據可讀性
- **邊框樣式**：標準的表格邊框，清晰分隔表頭和數據單元格
- **響應式容器**：支持水平滾動的響應式包裝器，適配各種屏幕尺寸
- **緊湊與標準模式**：提供不同的內邊距選項，適應不同的數據密度需求

### 交互體驗
- **懸停高亮**：行懸停時背景色變化，提供清晰的視覺反饋
- **可點擊行**：支持行點擊交互，配合指針樣式提示
- **排序功能**：表頭支持排序指示器，清晰表達數據排序狀態
- **響應式行為**：小屏幕下自動啟用水平滾動，保持數據完整性

### 適用場景
- 基於 Bootstrap 框架的快速項目開發
- 需要與現有 Bootstrap 組件庫統一風格的應用
- 企業內部管理系統和數據管理平台
- 多人協作的大型項目，需要標準化組件規範
- 原型設計和 MVP 產品的快速搭建

## English Version (en-US)

Please create a Bootstrap 5 style basic table component.

### Design Philosophy
Bootstrap 5 table continues the framework's pragmatic design tradition, providing professional out-of-the-box appearance and standardized component structure. Through mature design specifications and rich style variants, it offers developers a quick solution for building data tables, ensuring cross-project consistency and team collaboration efficiency.

### Visual Characteristics
- **Color Scheme**: Uses Bootstrap semantic color system with Bootstrap Blue as primary color, complemented by grayscale system
- **Striped Design**: Optional zebra striped rows alternating between white and light gray backgrounds, improving data readability
- **Border Style**: Standard table borders clearly separating headers and data cells
- **Responsive Container**: Responsive wrapper supporting horizontal scrolling, adapting to various screen sizes
- **Compact & Standard Modes**: Provides different padding options for varying data density needs

### Interaction Experience
- **Hover Highlight**: Background color changes on row hover, providing clear visual feedback
- **Clickable Rows**: Supports row click interactions with pointer cursor hints
- **Sorting Functionality**: Table headers support sort indicators clearly expressing data sorting state
- **Responsive Behavior**: Automatically enables horizontal scrolling on small screens, maintaining data integrity

### Use Cases
- Rapid project development based on Bootstrap framework
- Applications requiring unified style with existing Bootstrap component library
- Enterprise internal management systems and data management platforms
- Large collaborative projects needing standardized component specifications
- Rapid prototyping and MVP product development
