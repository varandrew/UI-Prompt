# Minimalism Basic Table

## 中文版本 (zh-CN)

請創建一個極簡主義風格的基礎表格組件。

### 設計理念
極簡主義表格追求"少即是多"的設計哲學，通過留白和對齊建立清晰的視覺秩序。去除一切非必要的裝飾元素，讓數據本身成為焦點。這種設計強調內容的可讀性和信息的清晰傳達，為用戶提供舒適、專注的數據瀏覽體驗。

### 視覺特徵
- **配色方案**：採用灰階色系，純白背景配合深灰文字，確保高對比度和可讀性
- **邊框設計**：極細的分割線或完全無邊框，依靠間距和對齊來區分數據區域
- **排版規範**：使用系統字體，適中的字號和充足的行高，確保長時間閱讀的舒適性
- **留白運用**：充足的單元格內邊距，保持視覺透氣感
- **行列設計**：清晰的表頭與數據行分離，可選的斑馬條紋提升可讀性

### 交互體驗
- **懸停反饋**：行懸停時背景色微妙變化，不破壞整體簡潔感
- **選中狀態**：使用單一品牌色高亮選中行，保持視覺一致性
- **排序指示**：簡潔的箭頭圖標，清晰表達排序狀態
- **響應式設計**：移動端自動切換為卡片式布局，保持數據完整性

### 適用場景
- 企業級數據管理系統和後台面板
- 需要長時間查看大量數據的專業應用
- 金融、財務報表等嚴肅場景
- 追求品牌一致性的現代企業應用
- 注重數據準確性和清晰度的科研平台

## English Version (en-US)

Please create a minimalism style basic table component.

### Design Philosophy
Minimalism table pursues the "less is more" design philosophy, establishing clear visual order through whitespace and alignment. By removing all unnecessary decorative elements, it makes the data itself the focal point. This design emphasizes content readability and clear information delivery, providing users with a comfortable and focused data browsing experience.

### Visual Characteristics
- **Color Scheme**: Uses grayscale colors with pure white background and dark gray text, ensuring high contrast and readability
- **Border Design**: Ultra-thin dividers or completely borderless, relying on spacing and alignment to distinguish data areas
- **Typography**: System fonts with moderate size and ample line height, ensuring comfort for prolonged reading
- **Whitespace**: Generous cell padding maintaining visual breathing space
- **Row & Column Design**: Clear separation between header and data rows, optional zebra stripes to improve readability

### Interaction Experience
- **Hover Feedback**: Subtle background color change on row hover without disrupting overall simplicity
- **Selected State**: Uses single brand color to highlight selected rows, maintaining visual consistency
- **Sort Indicators**: Simple arrow icons clearly expressing sorting state
- **Responsive Design**: Automatically switches to card layout on mobile, maintaining data integrity

### Use Cases
- Enterprise-level data management systems and backend panels
- Professional applications requiring prolonged viewing of large datasets
- Serious scenarios like finance and financial reports
- Modern enterprise applications pursuing brand consistency
- Scientific research platforms focusing on data accuracy and clarity
