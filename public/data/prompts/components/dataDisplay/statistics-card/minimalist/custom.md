# Minimalist Statistics Card

## 中文版本 (zh-CN)

請創建一個極簡主義風格的統計數據卡片組件。

### 設計理念
極簡主義統計卡片強調"少即是多"的設計哲學，通過精簡的視覺元素和清晰的信息層級，讓數據本身成為焦點。去除一切不必要的裝飾，用最純粹的方式呈現關鍵指標，為用戶提供一目了然的數據洞察體驗。

### 視覺特徵
- **配色方案**：採用中性色調為主，如純白背景搭配深灰文字，使用單一品牌色作為強調色突出重要數據
- **排版設計**：大號數字居中或左對齊，配合小號標籤文字，建立清晰的視覺層級
- **留白運用**：充足的內邊距和外邊距,讓內容呼吸，避免視覺擁擠
- **邊框與陰影**：淡灰色細線邊框或極淺的陰影，保持輕盈感
- **圖標設計**：線性圖標，與整體極簡風格保持一致

### 交互體驗
- **趨勢指示器**：使用簡潔的箭頭或百分比符號顯示數據變化趨勢
- **懸停效果**：微妙的背景色變化或陰影加深，不破壞整體簡潔感
- **過渡動畫**：平滑的淡入淡出效果，數字變化時採用緩動函數
- **響應式設計**：在不同屏幕尺寸下保持相同的視覺節奏和比例

### 適用場景
- 企業級儀表板和後台管理系統
- 數據分析平台的 KPI 展示
- 需要長時間專注查看數據的專業應用
- 追求品牌一致性的現代企業官網
- 金融、醫療等嚴謹行業的數據可視化

## English Version (en-US)

Please create a minimalist style statistics card component.

### Design Philosophy
The minimalist statistics card embodies the "less is more" philosophy, using streamlined visual elements and clear information hierarchy to make the data itself the focal point. By removing all unnecessary decoration, it presents key metrics in the purest way, providing users with an at-a-glance data insight experience.

### Visual Characteristics
- **Color Scheme**: Neutral tones dominate, such as pure white background with dark gray text, using a single brand color to highlight important data
- **Typography**: Large numbers centered or left-aligned, paired with small label text to establish clear visual hierarchy
- **Whitespace**: Generous padding and margins to let content breathe, avoiding visual clutter
- **Borders & Shadows**: Light gray thin borders or very subtle shadows to maintain a lightweight feel
- **Icon Design**: Linear icons that align with the overall minimalist aesthetic

### Interaction Experience
- **Trend Indicators**: Simple arrows or percentage symbols to show data change trends
- **Hover Effects**: Subtle background color changes or shadow deepening without disrupting overall simplicity
- **Transition Animations**: Smooth fade-in/fade-out effects, number changes use easing functions
- **Responsive Design**: Maintains the same visual rhythm and proportions across different screen sizes

### Use Cases
- Enterprise-level dashboards and backend management systems
- KPI displays for data analysis platforms
- Professional applications requiring prolonged data focus
- Modern corporate websites pursuing brand consistency
- Data visualization in rigorous industries like finance and healthcare
