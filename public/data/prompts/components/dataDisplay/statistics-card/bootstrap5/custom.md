# Bootstrap 5 Statistics Card

## 中文版本 (zh-CN)

請創建一個 Bootstrap 5 風格的統計數據卡片組件。

### 設計理念
Bootstrap 5 統計卡片延續了 Bootstrap 框架實用主義的設計傳統，強調開箱即用和跨平台一致性。通過標準化的組件結構和工具類系統，提供穩定可靠的數據展示方案，讓開發者能夠快速構建專業級的統計面板。

### 視覺特徵
- **配色方案**：使用 Bootstrap 5 語義化配色系統（primary、success、danger、warning、info），結合淺色和深色主題變體
- **卡片結構**：標準的卡片容器，包含清晰的內容區域，可選的頭部和底部區塊
- **圖標整合**：搭配圖標庫，使用彩色圖標背景圓形容器增強視覺識別
- **邊框與圓角**：標準圓角設計，可選的邊框色和陰影級別
- **網格系統**：響應式列布局，支持多種斷點的自適應排列

### 交互體驗
- **懸停狀態**：漸進式陰影變化，提供清晰的交互反饋
- **響應式行為**：自動適配移動端，從堆疊布局到水平網格的流暢過渡
- **顏色語義**：用綠色表示增長、紅色表示下降，符合用戶直覺
- **可訪問性**：內置無障礙標籤和語義化標記，確保屏幕閱讀器友好

### 適用場景
- 基於 Bootstrap 框架的項目快速開發
- 需要與現有 Bootstrap 組件庫統一風格的應用
- 內部管理系統和後台控制面板
- 多人協作的大型項目，需要標準化組件規範
- 原型設計和 MVP 產品的快速搭建

## English Version (en-US)

Please create a Bootstrap 5 style statistics card component.

### Design Philosophy
Bootstrap 5 statistics card continues the framework's pragmatic design tradition, emphasizing out-of-the-box usability and cross-platform consistency. Through standardized component structure and utility class system, it provides a stable and reliable data display solution, enabling developers to quickly build professional-grade statistical panels.

### Visual Characteristics
- **Color Scheme**: Uses Bootstrap 5 semantic color system (primary, success, danger, warning, info), combined with light and dark theme variants
- **Card Structure**: Standard card container with clear content areas, optional header and footer sections
- **Icon Integration**: Paired with icon libraries, using colored icon background circular containers to enhance visual recognition
- **Borders & Corners**: Standard rounded design with optional border colors and shadow levels
- **Grid System**: Responsive column layout supporting adaptive arrangement across multiple breakpoints

### Interaction Experience
- **Hover States**: Progressive shadow changes providing clear interaction feedback
- **Responsive Behavior**: Auto-adapts to mobile devices with smooth transition from stacked layout to horizontal grid
- **Color Semantics**: Green for growth, red for decline, aligning with user intuition
- **Accessibility**: Built-in accessibility labels and semantic markup ensuring screen reader friendliness

### Use Cases
- Rapid development of Bootstrap framework-based projects
- Applications requiring unified style with existing Bootstrap component library
- Internal management systems and backend control panels
- Large collaborative projects needing standardized component specifications
- Rapid prototyping and MVP product development
