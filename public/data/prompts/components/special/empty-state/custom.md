# Custom Prompt

## 中文版本 (zh-CN)

### 空状态

空状态组件，用于在没有内容可显示时显示

**分类**：特殊组件

**使用指南**：
此组件可以与不同的 UI 风格变体进行自定义。选择适合您设计系统的变体。


**实施注意事项**：
- 确保适当的可访问性属性
- 在不同屏幕尺寸上进行测试
- 验证颜色对比度以确保可读性
