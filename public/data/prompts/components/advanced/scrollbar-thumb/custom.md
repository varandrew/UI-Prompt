# Custom Prompt

## 中文版本 (zh-CN)

### 滚动条滑块

滚动条滑块组件，具有可自定义样式的滚动条手柄

**分类**：进阶组件

**使用指南**：
此组件可以与不同的 UI 风格变体进行自定义。选择适合您设计系统的变体。

**可用变体**：
- **极简现代**：极简现代滚动条滑块设计，带有简洁的样式
- **渐变效果**：渐变效果滚动条滑块设计，带有多彩渐变
- **品牌可定制**：品牌可定制滚动条滑块设计，支持品牌色彩定制
- **交互增强**：交互增强滚动条滑块设计，带有丰富的交互效果

**实施注意事项**：
- 确保适当的可访问性属性
- 在不同屏幕尺寸上进行测试
- 验证颜色对比度以确保可读性
