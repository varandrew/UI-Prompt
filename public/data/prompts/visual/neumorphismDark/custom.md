# Custom Prompt

## 中文版本 (zh-CN)

请使用 TailwindCSS 创建一个暗色 Soft UI / Neumorphism 界面：浅炭黑背景，大圆角卡片/按钮，内外双阴影，文字高对比。包含顶部导航、设定/控件区域（开关/滑块/输入）、KPI 卡与 CTA。阴影与高光保持克制，避免强霓虹或玻璃态。

---

## English Version (en-US)

Use TailwindCSS to build a dark Soft UI/Neumorphism page: charcoal base, large-radius cards/buttons, paired inner/outer shadows, high-contrast text. Include top nav, controls (toggles/sliders/inputs), KPI cards, and CTAs. Keep glow restrained—no heavy neon or glass.
