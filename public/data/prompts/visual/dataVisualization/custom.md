# Custom Prompt

## 中文版本 (zh-CN)

制作数据可视化仪表板：标题+筛选条，折线/柱状/饼或环图示例，KPI 卡，表格。颜色控制 5-7 色，网格细线、轴标题、单位清晰，Tooltip 简洁。

---

## English Version (en-US)

Create a data viz dashboard: header + filters, line/bar/pie(or donut) samples, KPI cards, table. Limit to 5–7 colors, fine grids, clear axes/units, concise tooltips.
