## Template ID Format

Template IDs should follow this pattern:
```
{category}-{familyId}-{variant}
```

Examples:
- `visual-grain-film-noir`
- `core-minimalism-saas-landing`
- `retro-artDeco-luxury`

For the default/main template in a family, use:
- `visual-grain` (just category-family)
- `core-minimalism`
